# ReQualifica+ | Global Solution 2025

## Descrição do Projeto

O ReQualifica+ é uma plataforma digital de requalificação profissional (SaaS - Software as a Service) projetada para auxiliar trabalhadores na transição de carreira em um mercado de trabalho cada vez mais automatizado e tecnológico.

A aplicação funciona como um "GPS de Carreira", utilizando inteligência artificial para analisar o perfil do usuário, identificar lacunas de competência (gaps) e gerar trilhas de aprendizado personalizadas. O sistema integra tecnologias modernas de front-end com lógica de recomendação, oferecendo uma interface intuitiva, acessível e responsiva.

### Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS** Para estilização e design do site.
* **JavaScript:** Para menu hamburguer, eventos e validação de formulário.

### Imagens e Ícones

* **Imagens:** Foram utilizadas imagens de alta resolução provenientes do Unsplash para ilustrar o contexto profissional e tecnológico.
* **Ícones:** Foram utilizados SVG como ícones direto no código para performance.

## Objetivo e Problema Resolvido

### O Problema
O mercado de trabalho enfrenta uma transformação acelerada devido à automação e Inteligência Artificial. Milhões de empregos estão sendo remodelados, criando um cenário onde a força de trabalho atual não possui as habilidades necessárias para as novas vagas. Isso gera um risco significativo de exclusão digital e desemprego estrutural, especialmente para grupos vulneráveis.

### O Objetivo
O ReQualifica+ tem como objetivo democratizar o acesso à requalificação profissional. A plataforma resolve o problema da "falta de direcionamento", conectando o perfil atual do trabalhador às demandas reais do mercado. Ela serve para guiar o usuário de forma estruturada, desde o diagnóstico de suas habilidades até a validação de novos conhecimentos, alinhando-se aos Objetivos de Desenvolvimento Sustentável (ODS) da ONU, especificamente Educação de Qualidade (4), Trabalho Decente (8) e Redução das Desigualdades (10).

## Estrutura de Pastas e Arquivos

A organização do projeto foi pensada para garantir facilidade de manutenção e leitura do código:

* **GS1-FRONT/** Pasta principal que contém toda a estrutura. Responsável por armazenar os HTMLs e as outras pastas
* **assets/**: Pasta destinada ao armazenamento de imagens locais e recursos de mídia estáticos utilizados no layout.
* **css/** Pasta com os arquivos CSS para a estilização do projeto.
* **js/:** Responsável por armazenar a validação de formulário, eventos e menu.
